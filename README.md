# 环湖骑行助手

青海湖 360km 逆时针骑行小助手。

## 在线使用

部署在 Vercel 上，手机/电脑直接打开链接即可使用：

> （部署后会自动更新此处链接）

- 账号：`cycle`
- 密码：`123456`

数据实时同步到 Supabase，两人共享同一份清单和住宿信息。

## 本地开发

```bash
cd qinghai-cycle
npm install
npm run dev
```

浏览器打开终端输出的地址即可。

## 功能

- **首页**：出发倒计时、今日行程
- **行程**：Day0-Day4 完整路书、关键节点、文化地理
- **装备**：公共 / Keith / Ada 分类勾选、实时统计
- **天气**：5 种骑行场景穿衣指南
- **住宿**：每晚酒店信息记录
- **我的**：骑行证书、安全须知

## 技术栈

Vue 3 + TypeScript + Vite + Tailwind CSS + Pinia + Supabase
